# Pong Game

A basic pong game for Red Academy students, using:

 HTML
 SVG
 Javascript
 Webpack

## Learning Objectives

Experiment with Javascript Object Oriented Programming and Webpack



## Setup

**Install dependencies:**

`> npm i`

**Run locally with Webpack Dev Server:**

`> npm start`

**Build for production:**

`> npm run build`

## Keys

**Player 1:**
* a: up
* z: down

**Player 2:**
* ▲ : up
* ▼: down
